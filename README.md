## 🌟 项目简介 (Overview)

`GrabTheP` 是一个零门槛、跨平台的交互式粒子物理实验室。它利用 **MediaPipe** 的视觉算法将你的手映射到 3D 虚拟空间，打破屏幕的物理限制，让你能直接“触摸”并重塑数字化身。

> [!TIP]
> **“抓个P”** 灵感来源于：**P**ixel (像素) → **P**articles (粒子) → **P**lay (玩耍)。

## ✨ 核心特性 (Features)

### 🚀 极致便捷
- **单文件架构**：整个项目封装在单个 `index.html` 中，无依赖包安装，双击即用。
- **低延迟 AI**：基于 WebGL 硬件加速，手势捕捉与物理计算实时同步。
- **导出录制**：内置 MediaRecorder 接口，支持录制 WebM 格式视频并一键下载。


### 🌊 五大运动场 (Engines)
- **三维正弦波**：跨越 XYZ 空间的波形拉伸，赋予图像流动生命。
- **动态涡旋中心**：漩涡中心随手而动，角速度随径向平滑衰减。
- **生命脉冲**：模拟生物心跳，握拳瞬间触发“狂暴模式”，感受脉动冲击。
- **分形噪声场**：基于伪随机算法的烟雾模拟，实现引力交互。
- **非牛顿物理**：首创粒子“剪切增稠”触感，挥手越快，空间粘度越大。

## 🎮 深度玩法指南 (Interactive Guide)

本项目不仅是视觉演示，更是一场深度的物理实验。

| 你的手势 (Input) | 动作描述 (Action) | 艺术反馈 (Output) |
| :--- | :--- | :--- |
| **✋ 张开手掌** | **慢速轻抚** | **引力吸附**：在噪声模式下，粒子会如灵动的绸缎缠绕你的指尖。 |
| **✋ 张开手掌** | **快速劈砍** | **物理排斥**：瞬间产生超强斥力场，将粒子群如海浪般整齐排开。 |
| **✊ 握紧拳头** | **原地保持** | **狂暴脉冲**：背景粒子波浪炸开，主体进入高频呼吸状态。 |
| **✊ 握紧拳头** | **虚空抓取** | **崩坏聚散**：粒子向掌心急速坍缩，随后在惯性中爆裂重组。 |
| **⚙️ 调节笔刷** | **Drag Brush** | 实时改变你的交互半径，体验从“微观触碰”到“上帝视角”的转换。 |

## 🚀 快速开始 (Quick Start)

1. **获取源码**：下载 `index.html`。
2. **启动实验**：在 **Chrome** 或 **Edge** 浏览器中打开文件（建议全屏）。
3. **注入灵魂**：点击左侧面板，上传一张图片（透明 PNG 效果最佳）。
4. **开启创作**：授权摄像头，挥动你的双手。

## 🛠️ 技术栈 (Tech Stack)

- **Rendering**: [Three.js](https://threejs.org/) (Custom ShaderMaterial)
- **AI Vision**: [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html)
- **Physics**: Custom CPU-based N-Body & Fluid Solver
- **Recording**: MediaRecorder API (VP9/WebM)
